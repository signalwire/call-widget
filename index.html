Introduction to the C2C widget

The C2C widget, short for "Click to Call widget" is a JavaScript widget which uses the SignalWire SDK and the SignalWire C2C service to allow you to very easily embed calls in your website. The calls can be routed to a phone number, a SIP address, custom AI agents, custom SWML scripts, and nearly infinite combinations of the above.

Usage of the C2C widget:

<!-- wherever in the html, your button -->
<button id="callButton">Call</button>

<!-- end of body -->
<c2c-widget
  buttonId="callButton"
  callDetails='{"destination":"/private/demo-1","supportsVideo":true,"supportsAudio":true}'
  token="<token>"
></c2c-widget>

<script src="https://assets.swrooms.com/c2c-widget.js" defer></script>

CDN URL​

The CDN URL is https://assets.swrooms.com/c2c-widget.js. This is a temporary URL, and will be put in cdn.signalwire.com later (maybe).

<script src="https://assets.swrooms.com/c2c-widget.js" defer></script>

But if you don't want to use the CDN, you can put the widget.js file wherever you want, and then include it like this:

<!-- near end of body -->
<script src="/path/to/widget.js" defer></script>
